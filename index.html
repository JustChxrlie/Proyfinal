Cual es el objetivo de CitasPoli?

Desarrollar un sistema de gestion de citas eficiente y confiable para mejorar la organizacion y la atencion en el control escolar de la Escuela Politecnica de Guadalajara, permitir a los usuarios programar citas de manera rapida y sencilla, Facilitar la gestion de citas por parte del personal administrativo y Proporcionar notificaciones automaticas para recordar a los usuarios sobre las citas programadas.

Funcionalidades Principales

  • Gestión y programación de citas.
  • Interfaz para visualizar citas registradas.
  • Consulta y cancelacion de citas.
  • Foro para usuarios que tengan dudas.
